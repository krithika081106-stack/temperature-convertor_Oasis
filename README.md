# 🌡️ Temperature Converter Web App

## 📌 Project Overview

This project is a **Temperature Converter Web Application** built using **HTML, CSS, and JavaScript**. It allows users to easily convert temperature values between **Celsius (°C), Fahrenheit (°F), and Kelvin (K)**.

The application features a **modern UI with animated gradients and dark/light mode toggle**, making it both functional and visually appealing.

---

## 🎯 Features

* 🌡️ **Temperature Conversion**

  * Celsius → Fahrenheit & Kelvin
  * Fahrenheit → Celsius & Kelvin
  * Kelvin → Celsius & Fahrenheit

* 🎨 **Animated Gradient Background**

  * Smooth color transitions for a dynamic look

* 🌙 **Dark / Light Mode Toggle**

  * Switch between light and dark themes with a single click

* ⚠️ **Input Validation**

  * Displays error message for invalid or empty input

* 💻 **Responsive Design**

  * Clean layout with glassmorphism card design

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling, animations, and UI design
* **JavaScript** – Logic for temperature conversion and theme toggle
